# Usage

Here is the basic usage guide:

* Do not ignore entries starting with .
    ```Shell
    lsx --all
    ```
* List directories before other files
    ```Shell
    lsx --group-directories-first
    ```
* List directories after other files
    ```Shell
    lsx --group-directories-last
    ```
* Show total entries count
    ```Shell
    lsx --show-total
    ```
* Show entry permissions
    ```Shell
    lsx --show-permissions
    ```
* Show entry owner
    ```Shell
    lsx --show-owner
    ```
* Show entry size
    ```Shell
    lsx --show-size
    ```
* Show entry last modified short date
    ```Shell
    lsx --show-date-modified
    ```
* Print help
    ```Shell
    lsx --help
    ```
* Print version
    ```Shell
    lsx --version
    ```
